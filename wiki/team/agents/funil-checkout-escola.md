# Funil Checkout Escola

> **Função:** Funil de checkout da escola: captura nome/email/telefone ANTES do Stripe (grava lead no Supabase pra habilitar recuperacao de carrinho), integra Stripe (checkout + webhooks) e dispara os eventos check
> **Categoria:** Outros / Customizados do nicho

## Como invocar

Você fala sempre com o CEO. Ele invoca este agente via:
```
Agent(subagent_type: "funil-checkout-escola", prompt: "<sua tarefa>")
```
