# Seo Flow

> **Função:** FLOW framework prompt analyst. Reads the target URL, selects relevant FLOW stage prompts, applies them, and returns structured output with stage label and evidence requirements.
> **Categoria:** SEO (busca orgânica + AEO/GEO)
> **Departamento:** [[../departments/seo|SEO (busca orgânica + AEO/GEO)]]

## Como invocar

Você fala sempre com o CEO. Ele invoca este agente via:
```
Agent(subagent_type: "seo-flow", prompt: "<sua tarefa>")
```
