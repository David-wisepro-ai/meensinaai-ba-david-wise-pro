# Seo Chief

> **Função:** Chief do Departamento SEO. Use quando a tarefa exige análise completa de site, auditoria técnica, SEO local, schema markup, Core Web Vitals, otimização para IA (ChatGPT/Gemini/Perplexity), SEO de e-co
> **Categoria:** Chiefs (orquestram squads)

## Como invocar

Você fala sempre com o CEO. Ele invoca este agente via:
```
Agent(subagent_type: "seo-chief", prompt: "<sua tarefa>")
```
