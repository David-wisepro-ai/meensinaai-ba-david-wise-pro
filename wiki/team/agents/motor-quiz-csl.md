# Motor Quiz Csl

> **Função:** Motor de simulado da escola de CSL (Construction Supervisor License, Massachusetts). Use pra montar quizzes por categoria (IRC/IBC/IECC/OSHA/AAB), montar provas completas de 70-75 questoes, controlar 
> **Categoria:** Outros / Customizados do nicho

## Como invocar

Você fala sempre com o CEO. Ele invoca este agente via:
```
Agent(subagent_type: "motor-quiz-csl", prompt: "<sua tarefa>")
```
