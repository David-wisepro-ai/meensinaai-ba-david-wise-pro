# Seo Image Gen

> **Função:** SEO image analyst. Audits existing OG/social preview images, identifies missing or low-quality images, and creates an image generation plan with prompts for key pages. Does NOT auto-generate images.
> **Categoria:** SEO (busca orgânica + AEO/GEO)
> **Departamento:** [[../departments/seo|SEO (busca orgânica + AEO/GEO)]]

## Como invocar

Você fala sempre com o CEO. Ele invoca este agente via:
```
Agent(subagent_type: "seo-image-gen", prompt: "<sua tarefa>")
```
