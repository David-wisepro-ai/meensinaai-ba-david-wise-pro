# Diego

> **Função:** Video Editor / Cortador especialista em Remotion + FFmpeg + Whisper. Use quando precisar transformar vídeo horizontal de YouTube em cortes verticais 9:16 pra Reels/TikTok/Shorts, transcrever áudio, id
> **Categoria:** Conteúdo
> **Departamento:** [[../departments/conteudo|Conteúdo]]

## Como invocar

Você fala sempre com o CEO. Ele invoca este agente via:
```
Agent(subagent_type: "diego", prompt: "<sua tarefa>")
```
