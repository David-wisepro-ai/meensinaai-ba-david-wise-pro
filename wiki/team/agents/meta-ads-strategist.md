# Meta Ads Strategist

> **Função:** Strategist do Squad Meta Ads. Use quando precisar decidir ESTRUTURA de campanha. Qual objetivo Meta (BRAND_AWARENESS, TRAFFIC, MESSAGES, OUTCOME_LEADS, OUTCOME_SALES), Advantage+ vs manual, bid strate
> **Categoria:** Meta Ads (anúncios pagos Meta)
> **Departamento:** [[../departments/meta-ads|Meta Ads (anúncios pagos Meta)]]

## Como invocar

Você fala sempre com o CEO. Ele invoca este agente via:
```
Agent(subagent_type: "meta-ads-strategist", prompt: "<sua tarefa>")
```
