# Plugin escola-csl

Agentes especializados na escola de licenca de construtor (CSL Massachusetts) da Wise Pro Academy.

| Agente | O que faz |
|---|---|
| motor-quiz-csl | Monta quizzes/provas a partir de questoes verificadas |
| gerador-questoes-csl | Gera+verifica questoes originais ancoradas no codigo |
| portal-aluno-builder | Constroi o portal multiusuario (login restrito) |
| builder-landing-escola | Site institucional + 3 landing pages (slot de VSL) |
| funil-checkout-escola | Captura pre-Stripe + Stripe + eventos |
| recuperacao-carrinho | Resend (email) + WhatsApp (Stevo/Z-API) |
| crm-escola-coordenador | Pipeline dos 3 produtos no Pedro |
| gbp-escola | Autoridade no Google / reviews |

MCP: `wisepro-mcp` (acesso ao Supabase).
